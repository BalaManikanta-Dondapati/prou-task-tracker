# ProU Task Tracker

A small full-stack Task Tracker built for ProU technology intern assessment.  
Frontend: React + Vite. Backend: Node.js + Express + SQLite. Authentication: JWT.

## Features
- CRUD operations for Tasks and Employees
- JWT authentication (signup/login)
- Protected endpoints for create/update/delete tasks
- Responsive frontend with a clean UI
- Filters for tasks by status and assignee
- Bonus: Authentication, polished UI, deployment instructions

---

## Tech Stack
- Frontend: React, Vite, Axios, React Router, Tailwind (development)/CSS for polish
- Backend: Node.js, Express, SQLite
- Auth: JWT (jsonwebtoken), password hashing (bcryptjs)
- Dev tools: nodemon, dotenv
